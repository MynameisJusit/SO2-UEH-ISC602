¿Qué son las Pilas

Una pila es una estructura de datos lineal que organiza elementos siguiendo el principio LIFO (Last In, First Out), que significa “el último en entrar es el primero en salir”. Esto quiere decir que el último elemento agregado a la pila será el primero en eliminarse.

Operaciones con las Pilas

Push: Agregar un elemento.

Pop: Eliminar un elemento superior.

Peek o Top: Consultar el elemento superior sin eliminarlo.

Representación

La pila se representa como una estructura vertical donde las inserciones y eliminaciones se realizan únicamente en la parte superior llamada tope.

Ejemplo de pila:

								
															 Tope
															   ↓
															┌─────┐
															│ 40  │
															├─────┤
															│ 30  │
															├─────┤
															│ 20  │
															├─────┤
															│ 10  │
															└─────┘

En este ejemplo:

  • El número 40 es el último elemento agregado.
  • El número 10 fue el primero en entrar.
  • Si se aplica una operación Pop, el número 40 será eliminado primero.
