Definición

El FTP (File Transfer Protocol) es un protocolo de red utilizado para transferir archivos entre sistemas conectados a través de una red, como Internet.
Funciona bajo el modelo cliente-servidor y permite subir, descargar y administrar archivos de manera remota.

Cliente FTP

Es el programa o aplicación que utiliza el usuario para conectarse a un servidor FTP.

  • Permite subir archivos al servidor
  • Permite descargar archivos
  • Permite organizar carpetas remotas

Ejemplo: FileZilla, WinSCP

Servidor FTP

Es el sistema que almacena los archivos y responde a las solicitudes del cliente.

  • Administra permisos de acceso
  • Gestiona usuarios y contraseñas
  • Controla la transferencia de datos

Funcionamiento

  1. El cliente solicita conexión al servidor.
  2. El usuario se autentica (usuario y contraseña).
  3. Se establece la conexión de control.
  4. Se inicia la transferencia de archivos.

FTP utiliza normalmente el puerto 21 para la conexión.

Seguridad en FTP

El FTP tradicional no cifra la información.

Por ello existen versiones más seguras como:

  • FTPS (FTP Secure)
  • SFTP (SSH File Transfer Protocol)
